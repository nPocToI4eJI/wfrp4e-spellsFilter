# WFRP4e - Spells Filter
Этот модуль позволит группировать заклинания по пользовательским группам и отображать в списке только необходимые.

При включении этого модуля, в листе привязанного к игроку актёра, во вкладке **"Магия"**, будет отображаться кнопка фильтра:  
<p align="center">
  <img width="750" height="30" alt="image" src="https://github.com/user-attachments/assets/6b127eed-a9fd-479d-bf15-bc1d12e3b475"/>
</p>

- При нажатии **ЛКМ** фильтр переключается на следующий.  
- При нажатии **ПКМ** открывается меню, где Вам будет предложено выбрать Заклинания, которые будет отображать текущий фильтр.

Доступные фильтры:
- **Все** (_отображаются все Заклинания, без возможности настройки_)
- **Боевые**
- **Исследование**
- **Другое**

Кнопка фильтра отображается только для Актёра, который является Персонажем игрока.
